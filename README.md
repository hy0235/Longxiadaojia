## 🌟 项目简介

**虾装宝 (ClawMaster)** 是一款参照“东郊到家”模式打造的专业上门安装与调优平台。我们专注于为用户提供 **OpenClaw** 的一站式部署服务。

通过平台，用户可以预约当地资深的“调优师（Tuner）”上门进行环境配置、Docker 部署及性能优化。

### 🎨 设计调性
*   **视觉风格**：采用经典的“蒂芙尼绿 (Tiffany Green)”配色，清新、专业且极具亲和力。
*   **用户体验**：极简的卡片式布局，流畅的交互动画，完美复刻本地生活服务类 App 体验。

## ✨ 核心功能

-   **📦 调优方案**：从远程轻量调试到带硬件上门部署，多种套餐满足不同需求。
-   **📡 雷达发现**：实时“扫描”附近的资深调优师，查看其“土味且专业”的极客档案。
-   **📟 控制台**：直观的订单管理系统，随时掌握部署进度。
-   **🛠️ 商户中心**：支持技术极客入驻，缴纳保证金后即可开启“深夜部署”生涯。

## 🛠️ 快速开始

### 1. 克隆项目
```bash
git clone https://github.com/your-username/claw-master.git
cd claw-master
```

### 2. 安装依赖
```bash
npm install
```

### 3. 配置环境变量
在根目录创建 `.env` 文件，并填入你的 Gemini API Key：
```env
VITE_GEMINI_API_KEY=你的API密钥
```

### 4. 启动开发服务器
```bash
npm run dev
```

## 🌐 部署到 Vercel

1.  将代码推送至 GitHub。
2.  在 Vercel 中导入该仓库。
3.  在 **Environment Variables** 中添加 `VITE_GEMINI_API_KEY`。
4.  点击 **Deploy**，即刻开业！
